This project recommends movies based on similarity scores using cosine similarity.  
It uses a dataset of movies and is built using Python, Pandas, and scikit-learn.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`

## Files
- `app.py` : Main application logic
- `df.pkl` : Preprocessed movie data
- `requirements.txt` : Dependencies
